# 🚀 Quick Fix: Kakao Maps Not Loading

## The Problem
Map shows error: **"지도를 불러올 수 없습니다"** (Cannot load map)

## The Solution (5 minutes)

### 1️⃣ Go to Kakao Developers
```
https://developers.kakao.com
```

### 2️⃣ Login → Click "앱" (App)

### 3️⃣ Select your app → "앱 설정" → "플랫폼"

### 4️⃣ Click "Web 플랫폼 등록" (Register Web Platform)

### 5️⃣ Enter this domain:
```
http://localhost:3000
```

### 6️⃣ Click "저장" (Save)

### 7️⃣ Refresh your browser
```
http://localhost:3000
```

## ✅ Done!
The map should now load successfully.
